# タイピング音 サウンドクレジット

このフォルダに置く音声ファイルは、各テーマのキー入力音として再生されます。
ファイルが無い場合は、コードが自動的に合成音(Web Audio)へフォールバックするため、
アプリは音源未配置でも正常に動作します。

## 配置するファイル

各テーマに対応する短い1音(目安: 50〜300ms、mp3)を、以下のファイル名で置いてください。

| テーマ       | ファイル名        | 想定する音 |
|--------------|-------------------|------------|
| (既定)       | `defult.mp3`      | テーマ固有のファイルが無い/読めないときに使う共通の既定音 |
| コトコト     | `kotokoto.mp3`    | メカニカルキーの打鍵音(クラッキー/リニア系) |
| もすもす     | `mosumosu.mp3`    | 静音・柔らかい打鍵音(サイレント系) |
| ぷにゅぷにゅ | `punyupunyu.mp3`  | 柔らかいぷにゅっとした音 |
| バブル       | `bubble.mp3`      | 泡がはじけるポップ音 |
| 洞窟         | `doukutu.mp3`     | 洞窟内の反響する打音 |
| 石           | `stone.mp3`       | 石を打つ硬い音 |
| にゃーにゃー | `nyanya.mp3`      | 猫の短い鳴き声 |
| わんわん     | `wanwan.mp3`      | 犬の短い鳴き声 |
| ぴよぴよ     | `piyopiyo.mp3`    | ひよこ・小鳥の短い鳴き声 |

- 1音は短く、無音の前後をトリミングしておくと反応が良くなります。
- アプリは再生時に playbackRate を僅かに揺らすため、同じ音でも単調になりません。
- 再生の優先順位は「テーマ固有ファイル → `defult.mp3` → 合成音」。`defult.mp3` が
  あれば、テーマ固有ファイルが無いときもこの既定音が鳴ります。

## ライセンスの記録(必須)

ダウンロードした各ファイルの出所とライセンスを、必ずここに追記してください。
使用は **Pixabay(帰属不要・商用可・無料)** または **Freesound の CC0(パブリックドメイン)**
のものに限定します。KOKIMIKEYS など他アプリのアセットを直接転用しないこと。

例:

```
- kotokoto.mp3
  - 出所: Pixabay (https://pixabay.com/sound-effects/xxxxx)
  - 作者: <uploader>
  - ライセンス: Pixabay Content License (帰属不要・商用可)
```
